# Excel版本比对

## 功能
比对2个excel表格，并分析指定范围内的表格差异，在其中1个表格高亮差异单元格，同时生成1个变化列表，变化列表如下：

| 序号 | 中文名称 | 变更前 | 变更后 | 变更时间 |
| ---- | -------- | ------ | ------ | -------- |
|      |          |        |        |          |
|      |          |        |        |          |
|      |          |        |        |          |

比对前对表格分别进行检查：

1. 比对两表格标题行是否一致：

2. 分别进行**行状态**检查：

   - 第1列为序号列，其他列为内容列

   - 序号存在、内容列中必需列均不为空，状态OK
   - 序号存在、内容列中必需列存在空值，状态NOK

3. 进行行比对：

4. 对存在行变更的行再进行单元格比对
